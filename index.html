webpay-magento

Magento Payment Gateway Extension for WebPay integration.

A how to Guide

Manual Installation

1. Download package as .zip or .tar
2. Upload to your maagento's website root folder.
3. Extract the uploaded package.
4. Flush cache in magento admin panel.

Automated Installation (via Magento Connect)

1. Go to “System => Magento Connect => Magento Connect Manager”, use your Magento admin account to login.
2. In “Direct package file upload” area, choose payment zip file, then Upload it, it will install payment module into your Magento site.

Setting up the module

1) Go to payment methods page: “System => Configuration”, then “Payment Methods” in the left.
2) Find “Credit Card (via MCPayment)”, click it to open.
3) Enable payment module.
4) Choose Recurring to Yes if you want recurring payment.
5) Select order status for order before paying on MCpayment website, usually ‘pending’.
6) Input ‘merchant ID’, ‘secure key’, these are from your MCpayment merchant account.
7) Choose Transaction Mode ‘live’ if you want to make live transactions.
8) Input ‘sort order’ number, this is for payment ordering in payment list at front-end, when customer checkout.
